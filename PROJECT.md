# 🚀 Project Overview

## 🧩 О проекте

Разрабатывается одностраничный промо-сайт (landing page) для презентации технологического проекта китайской аудитории.

Основная цель:

- презентация продукта
- донесение ценности
- сбор лидов

Стиль: clean-tech / minimalism (ориентир: graphenea.com)

---

## 🌐 Требования

- Одностраничный сайт
- Адаптивность (desktop / tablet / mobile)
- Языки: English, Chinese
- Быстрая загрузка
- SEO

---
